Pengenalan HTML

HTML adalah bahasa pemrograman dalam membangun website.HTML merupakan singkatan dari HyperText Markup Language Ada berbagai macam tag elemen yang dapat digunakan seperti heading dan paragraf

dalam membuat halam web,HTML dapat dikolabarisikan dengan bahasa lainnya seperti java script dan css (cascading style sheet).CSS adalah bahasa khusus yang digunakan untuk memperindah tampilan web
